# Search Engines for CyberSecurity

A comprehensive list of specialized search engines for Open Source Intelligence (OSINT), reconnaissance, and vulnerability research.

---

### Internet of Things (IoT) & Devices
* **Shodan**: Search for devices connected to the internet (routers, cams, SCADA).
* **Censys**: Analyzes the "attack surface" of the internet to find hosts and networks.
* **ZoomEye**: Cyberspace search engine for identifying devices and web components.
* **Wigle**: Database of wireless networks (Wi-Fi and cell towers) with GPS data.
* **GreyNoise**: Filters "internet background noise" to see who is actively scanning.
* **Onyphe**: A cyber defense search engine for open-source and proprietary intelligence.
* **Fofa**: A search engine for mapping the cyberspace of assets.

### Domain, DNS & Infrastructure
* **Crt.sh**: Certificate Transparency logs—excellent for finding subdomains.
* **ViewDNS.info**: A massive toolkit for DNS lookups, reverse IP, and port scans.
* **DNSDumpster**: A domain reconnaissance tool to map an organization's DNS records.
* **WhoisXML API**: Lookup registration records and historical WHOIS data.
* **SecurityTrails**: Explore complete historical DNS data and IP history.
* **Spyse**: Search for domains, IP addresses, and organizational infrastructure.

### Code, Leaks & Vulnerabilities
* **Exploit-DB**: Archive of exploits and shellcodes for security researchers.
* **Vulners**: Searchable database for CVEs, exploits, and security advisories.
* **Searchcode**: Search for functions or sensitive info (like API keys) in public code.
* **PublicWWW**: Search the HTML, JS, and CSS source code of millions of websites.
* **LeakIX**: Search for publicly exposed services and leaked data.
* **BinaryEdge**: Real-time threat intelligence and attack surface monitoring.

### Files & Malware Analysis
* **VirusTotal**: Analyze files and URLs for viruses, trojans, and malicious behavior.
* **Hybrid Analysis**: Free malware analysis tool providing sandboxed execution.
* **Any.Run**: Interactive online malware sandbox.
* **Joe Sandbox**: Deep malware analysis for Windows, Android, Mac, and iOS.
* **Kintel**: Intelligence search for malware indicators and samples.

### OSINT & Dark Web
* **Intelligence X (IntelX)**: Search the darknet, data leaks, and historical web archives.
* **PimEyes**: Reverse image search to find where faces appear online.
* **Have I Been Pwned**: Check if an email address or phone has been in a data breach.
* **Hunter.io**: Find professional email addresses associated with any domain.
* **TinEye**: Reverse image search for tracking the source of an image.
* **Shroom**: Search engine for investigating darknet marketplaces and forums.
* **Epieos**: Find information linked to email addresses without notifying the user.
* **RocketReach**: Search for contact information (emails and social profiles).

---

> **Note:** Use these tools responsibly. They are intended for security research, defensive mapping, and authorized penetration testing.
